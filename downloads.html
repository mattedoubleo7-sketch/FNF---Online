Quick Guide

Which One Should Friends Use?

  • Best choice: send friends the hosted browser URL and let them click Play Online In Browser.
  • Online: use this if friends are playing from different houses or different PCs.
  • Offline: use this if you just want local play with no server, or if two players are sharing one computer.
  • App: for now, use the browser install option on the hosted page. The direct hosted EXE download is not on Render yet.
  • JS: use this if you want the source files or want to self-host the build yourself.
